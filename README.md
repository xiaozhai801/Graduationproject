# 基于javaweb的物品修复系统的设计与实现
基于jsp+servlet的MVC开发模式物品修复系统

系统分为前端面向用户的物品修复系统和面向管理的管理系统。

前端系统需求分析：

1.	用户注册功能;
2.	用户登录功能;
3.	用户信息修改功能;
4.	浏览物品修复文章功能;
5.	评论功能;
6.	评论管理功能;
7.	文章分类显示;
